Features

Boiler allows to control kettles remotely using Bluetooth LE connection.

Supported devices

  • Redmond Skykettle
    • RK-G200S, RK-G210S, RK-G211S (tested)
    • other 2nd-gen devices (RK-G2xx) (not tested)
